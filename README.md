# Тема
Интернет-магазин

# Выбранная сущность
В этой лабораторной работе реализован класс `Product`, который описывает товар интернет-магазина.

## Описание класса

Класс `Product` содержит:

- название товара;
- цену;
- количество на складе;
- категорию;
- скидку;
- состояние товара.

## Атрибуты класса

- `shop_name` - название магазина;
- `max_discount` - максимальная скидка.

## Закрытые атрибуты экземпляра

- `_name`
- `_price`
- `_stock`
- `_category`
- `_discount`
- `_is_active`

## Что реализовано

В классе реализованы:

- конструктор с проверкой входных данных;
- свойства `@property`;
- `setter` для цены и скидки;
- отдельные методы валидации;
- методы `__str__`, `__repr__`, `__eq__`;
- бизнес-методы:
  - `final_price()`
  - `add_stock()`
  - `buy()`
  - `deactivate()`

## Проверки данных

В программе проверяется:

- тип данных;
- пустая строка;
- цена больше 0;
- количество не отрицательное;
- скидка в допустимом диапазоне.

## Логическое состояние объекта

Товар может быть активным или неактивным.

Реализовано:

- если товара не осталось, он становится неактивным;
- неактивный товар нельзя купить;
- состояние можно изменить методом `deactivate()`.

## Что показано в `demo.py`

В демонстрационном файле показаны:

- создание объекта;
- вывод объекта;
- сравнение объектов;
- изменение цены и скидки через `setter`;
- доступ к атрибуту класса через класс и экземпляр;
- покупка товара;
- пополнение товара;
- изменение состояния;
- обработка ошибок через `try/except`.
